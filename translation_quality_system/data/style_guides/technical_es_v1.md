# Unitron English-to-Spanish Technical Translation Style Guide

Use professional neutral Spanish suitable for Spain and for hearing-aid user guides, fitting-software manuals, medical-device documentation, and technical support material.

## Mandatory preservation rules

- Preserve all numbers, versions, units, URLs, trademarks, model names, placeholders, tags, and product names exactly unless the official Spanish reference clearly changes them.
- Preserve protected product and protocol terms such as Unitron, Sonova, Bluetooth®, BT-LE, GMS, iOS, Android OS, Moxi, TrueFit, FLEX:TRIAL, TV Connector, Remote Plus, Remote Control, PartnerMic, Roger, Noahlink Wireless, iCube II, RECD, REUG, WDRC, NAL-NL1, NAL-NL2, and DSL v5.
- Do not silently change compatibility versions such as iOS version, Android OS version, Bluetooth version, frequency ranges, time durations, or battery size.

## Preferred terminology

- Translate "hearing aid(s)" and "hearing instrument(s)" as "audífono(s)".
- Translate "hearing care professional" as "profesional de salud auditiva".
- Translate audiology "fitting" as "adaptación", not "ajuste", when referring to fitting sessions, fitting formulas, or fitting software.
- Translate "fitting software" as "software de adaptación".
- Translate "battery door" as "portapilas" and "battery/batteries" as "pila/pilas" in the device-manual context.
- Translate "flight mode" as "modo avión".
- Translate "pairing mode" as "modo de emparejamiento".
- Translate "drop-down menu" as "menú desplegable".
- Translate "pop-up window" as "ventana emergente".

## UI style

- Use formal imperative where the source gives instructions: "Pulse", "Seleccione", "Mantenga pulsado", "Consulte".
- Avoid leaving English UI verbs such as "Click", "Press", "Tap", "Print", "Save", "Undo", "Redo", "Update", "Apply", "Cancel", "Detect" unless they are official untranslated product labels.
- Use Spanish opening punctuation: ¿ and ¡.

## Safety and regulatory content

- Never omit negation or prohibition in safety warnings, especially sentences with "never", "must", "required", "warning", "do not", "flight mode", water exposure, batteries, medical-device compliance, or tinnitus settings.
- If a segment has missing numbers, changed versions, hallucinated medical content, repeated loops, untranslated English chunks, or broken safety instructions, route it to human review instead of auto-accepting it.
